# Simulador de Streaming de Música

Sistema de classificação de áudios que simula uma plataforma de streaming, com músicas e podcasts com base em POO Java.

## 🧠 Conhecimentos utilizados
- Java
- POO

## 📋 Funcionalidades
- Cadastrar áudio —> define título, duração e outras informações de músicas ou podcasts
- Reproduzir —> incrementa o contador de reproduções do áudio
- Curtir —> incrementa o contador de curtidas
- Classificar automaticamente —> atribui nota 10 ou 7/8 com base nas reproduções (músicas) ou curtidas (podcasts)
- Avaliar preferência —> verifica se o áudio é "sucesso absoluto" ou "boa opção para depois" com base na classificação.

## ▶️ Como executar
1. Clone o repositório
2. Abra no Eclipse
3. Execute a classe Main
